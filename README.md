# Auto URL Opener (自動網頁開啟工具)

這是一個輕量級、純靜態的網頁工具，旨在幫助使用者依序、定時開啟大量的網址清單。
適合處理大量連結檢查、簽到或瀏覽作業。

## ✨ 主要功能 (Features)

* **🛡️ 隱私優先**：純 HTML/JS 前端運作，無後端伺服器，不追蹤也不上傳任何使用者資料。
* **🧹 智慧清理**：複製貼上即可！自動辨識網址，忽略行首的編號（如 `1.`、`2、`）或文字說明。
* **🌙 深色模式**：內建護眼深色主題 (Dark Mode)，並會自動記憶您的偏好設定。
* **⏱️ 定時控制**：可自訂開啟間隔（秒數），避免瞬間開啟過多視窗導致電腦卡頓。
* **🛑 自動停止**：清單執行完畢後自動停止，並提供「一鍵關閉剛剛開啟的所有分頁」功能。

## 🚀 如何使用 (How to use)

1. **開啟網頁**：直接進入線上連結。
2. **輸入網址**：將整串網址清單貼入文字框（不需要手動刪除編號）。
3. **設定間隔**：輸入您希望的間隔秒數（預設 120 秒）。
4. **開始**：點擊「開始執行」。
5. **權限設定**：首次使用若瀏覽器阻擋，請點擊網址列右側圖示，選擇「允許開啟彈跳視窗」。

## 🛠️ 技術細節

* **Core**: HTML5, CSS3, Vanilla JavaScript (ES6+)
* **Style**: CSS Variables (for theming), Responsive Design
* **Storage**: LocalStorage (for saving theme preference)
